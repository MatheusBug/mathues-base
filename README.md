# Matheus Amorim | Marketing Jurídico & Autoridade Digital

Desenvolvido por **Matheus Amorim** (https://github.com/MatheusBug).

## Sobre o Projeto

Este repositório contém o código-fonte do meu portfólio profissional e landing page de alta conversão, focado em **Marketing Jurídico e Autoridade Digital**. 

O projeto foi arquitetado para transmitir sobriedade, confiança e profissionalismo, atendendo especificamente advogados e profissionais de áreas reguladas (como saúde e engenharia) que buscam posicionamento estratégico na internet em conformidade com normas éticas (como o Provimento 205/2021 da OAB).

### Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias mais modernas de desenvolvimento web para garantir performance, SEO e escalabilidade:

- **React 19** com **TypeScript**: Para uma arquitetura robusta e tipada.
- **Vite**: Build tool de alta performance.
- **Tailwind CSS**: Para estilização utility-first, responsiva e com design system customizado (Cores Premium).
- **Lucide React**: Ícones leves e elegantes.
- **React Router DOM**: Para navegação SPA (Single Page Application).
- **SEO Avançado**: Otimização de meta tags, Open Graph e JSON-LD para indexação no Google.

### Funcionalidades Principais

- **Design Premium & Responsivo**: Layout fluido que se adapta a qualquer dispositivo, com estética "High-End".
- **SEO Otimizado**: Estrutura semântica e meta-dados dinâmicos para ranqueamento local e orgânico.
- **Navegação Intuitiva**: Menu com scroll suave e links diretos.
- **Compliance Ético**: Textos e estruturas pensadas para respeitar as limitações de publicidade da OAB.
- **Contato Direto**: Integração com WhatsApp e Email para conversão de leads.

## Como Executar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/MatheusBug/mathues-base.git
```

2. Instale as dependências:
```bash
npm install
```

3. Execute o servidor de desenvolvimento:
```bash
npm run dev
```

4. O projeto estará rodando em `http://localhost:5173`.
